# AnnoncesNcFilter

Simple Python script to fetch and print the title of `https://annonces.nc`.

## Setup

1. Create a virtual environment:
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```
2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```

## Run

```powershell
python main.py
```
